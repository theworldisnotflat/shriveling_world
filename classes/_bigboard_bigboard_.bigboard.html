Class BigBoard

C'est la classe qui contrôle toute l'application: la liste des cônes, pays et courbes ainsi que la scene THREE.JS + les commandes et le comportement...

This class controls all the application:

  • the list of _cones,
  • _countries,
  • curves This is where the THREE.JS scene is defined with commands and behaviors
